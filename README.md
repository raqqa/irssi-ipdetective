### Dependencies

#### GeoIP C API
In Debian/Ubuntu,
```
sudo apt-get install libgeoip-dev
```

#### Geo::IP perl module
Follow instructions at [http://search.cpan.org/~maxmind/Geo-IP-1.45/] http://search.cpan.org/~maxmind/Geo-IP-1.45/

OR

```
sudo cpan
install Geo::IP
```

#### Regexp::Common::net perl module

```
sudo cpan
install Regexp::Common::net
```
